发表于2021-06-18|更新于2021-06-18|hexo手册
|字数总计:2.9k|阅读时长:11分钟

修改站点配置文件 _config.yml | _config.butterfly.yml

网站资料

  • 修改网站各种资料，例如标题、副标题和邮箱等个人资料，请修改博客根目录的_config.yml

语言

  • 默认语言是 en
  • 主题支持三种语言

导航菜单

  • 必须是 /xxx/，后面||分开，然后写图标名，如果不希望显示图标，图标名可不写

代码

代码高亮主题

代码复制

代码框展开/关闭

  • 默认情况下，代码框自动展开，可设置是否所有代码框都关闭状态，点击>可展开代码

代码换行

  • 在默认情况下，Hexo 在编译的时候不会实现代码自动换行

代码高度限制

  • 可配置代码高度限制，超出的部分会隐藏，并显示展开按钮

复制设置

  • 可配置网站是否可以复制、复制的内容是否添加版权信息

社交图标

  • Butterfly支持 font-awesome v5图标. 图标素材
  • 书写格式 图标名：url || 描述性文字

文章页相关配置

主页文章节选

  • 在butterfly里，有四种可供选择
  1. description： 只显示description
  2. both： 优先选择description，如果没有配置description，则显示自动节选的内容
  3. auto_excerpt：只显示自动节选
  4. false： 不显示文章内容

    description在front-matter里添加 description: 章节简述

图片展示设置

  • 配置的值
  1. 留空 显示默认的top_img（如有）文章页top_img留空的话，会显示 cover 的值
  2. img链接 图片的链接，显示所配置的图片
  3. 顔色(#0000FF rgb(0,0,255) 对应的顔色
  4. transparent 透明
  5. false 不显示 top_img

文章置顶

  • 可以直接在文章的front-matter区域里添加sticky: 1属性来把这篇文章置顶。数值越大，置顶的优先级越大

文章封面

  • 文章的markdown文档上,在Front-matter添加cover,并填上要显示的图片地址。
  • 如果不配置cover,可以设置显示默认的cover.如果不想在首页显示cover,可以设置为false

是否显示文章封面

文章封面的位置

  • 三个值可配置 left , right , both

当没有设置cover时，默认的封面显示

相册

主题封面

  • name：图库名字
  • description：图库描述
  • link：连接到对应相册的地址
  • img-url：图库封面的地址

相册内容

tag-hide

如果你想把一些文字、内容隐藏起来，并提供按钮让用户点击显示

inline

  • 在文本里面添加按钮隐藏内容，只限文字
  • content: 文本内容
  • display: 按钮显示的文字(可选)
  • bg: 按钮的背景颜色(可选)
  • color: 按钮文字的颜色(可选)

block

  • 独立的block隐藏内容，可以隐藏很多内容，包括图片，代码块
  • content: 文本内容
  • display: 按钮显示的文字(可选)
  • bg: 按钮的背景颜色(可选)
  • color: 按钮文字的颜色(可选)

toggle

  • 如果你需要展示的内容太多，可以把它隐藏在收缩框里，需要时再把它展开

mermaid

使用mermaid标籤可以绘製 Flowchart（流程图）、Sequence diagram（时序图 ）、Class Diagram（类别图）、State Diagram（状态图）、Gantt（甘特图）和Pie Chart（圆形图）

详解：

  • https://github.com/knsv/mermaid
  • https://mermaid-js.github.io/mermaid/

Tabs

Button

  • [url] : 链接
  • [text] : 按钮文字
  • [icon] : [可选] 图标
  • [color] : [可选] 按钮背景顔色(默认style时） 按钮字体和边框顔色(outline时) default/blue/pink/red/purple/orange/green
  • [style] : [可选] 按钮样式 默认实心 outline/留空
  • [layout] : [可选] 按钮佈局 默认为line | block/留空
  • [position] : [可选] 按钮位置 前提是设置了layout为block 默认为左边| center/right/留空
  • [size] : [可选] 按钮大小| larger/留空

inlineImg

  • 主题中的图片都是默认以块级元素显示，如果你想以内联元素显示
  • [src] : 图片链接
  • [height] ： 图片高度限制【可选】

label

  • 高亮所需的文字
