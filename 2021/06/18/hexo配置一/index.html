Hexo配置一

|字数总计:1.4k|阅读时长:5分钟

站点目录文件的基本配置

Butterfly

更换主题

  1. 直接在github链接上下载下来，然后放到theme文件夹下
  2. 在配置文件中把theme换成那个主题文件夹的名字，它就会自动在 主题 文件夹中搜索你配置的主题
  3. 而后进入 主题 这个文件夹，可以看到里面也有一个配置文件_config.yml，这个配置文件是修改你整个主题的配置文件。

安装

  • 打开Git到你的博客根目录里

应用主题

  • 修改站点配置文件_config.yml，把主题改为butterfly theme: butterfly

安装插件

  • 如果你没有 pug 以及 stylus 的渲染器，下载安装

升级

  • Git根目录下输入
  • 把主题文件夹中的 _config.yml 复製到 Hexo 根目录里，同时重新命名为 _config.butterfly.yml

Front-matter

  • Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量

Page Front-matter

Post Front-matter

标签页

  • 输入 hexo new page tags
  • 你会找到source/tags/index.md这个文件

分类页

  • 输入 hexo new page categories
  • 你会找到source/categories/index.md这个文件

友情链接

  • 输入 hexo new page link
  • 你会找到source/link/index.md这个文件
  • 友情链接添加

    在Hexo博客目录中的source/_data（如果没有 _data 文件夹，请自行创建），创建一个文件link.yml
    /* class_name和class_desc支持html格式书写，如不需要，也可以留空。*/

图库

  • 图库页面只是普通的页面，你只需要 hexo n page xxxxx 创建你的页面就行
  • 然后使用标签外挂 galleryGroup，具体用法请查看对应的内容

    子页面

  • 子页面也是普通的页面，你只需要hexo n page xxxxx 创建你的页面就行

404页面

  • 主题内置了一个简单的404页面，可在设置中开启
  • 如需本地预览，请访问 http://localhost:4000/404.html
